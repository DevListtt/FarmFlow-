# FarmFlow — Identité graphique & direction UI

FarmFlow doit avoir une identité visuelle moderne, sobre, technologique et professionnelle.

L'objectif n'est pas de créer une application agricole “verte / bio / nature”, mais un cockpit numérique sérieux pour piloter une exploitation agricole moderne.

## Positionnement visuel

FarmFlow doit évoquer le pilotage, la donnée, la fluidité, la fiabilité, la performance, la clarté, l'indépendance numérique et la modernisation des fermes.

FarmFlow ne doit pas ressembler à une marque bio, une coopérative agricole traditionnelle, une application verte générique, un outil administratif daté, un logiciel agricole surchargé ou une interface générée sans direction artistique.

## Logo / wordmark

Le logo principal de FarmFlow est typographique.

Le wordmark principal est : **FarmFlow**.

Direction :

- “Farm” en bleu marine profond `#0A132B` ;
- “Flow” en bleu électrique / cobalt `#2563EF` ;
- typographie moderne, arrondie, fluide et professionnelle ;
- style proche de Manrope ExtraBold / Manrope Bold ;
- pas d'effet 3D ;
- pas d'illustration agricole excessive ;
- pas de vert dominant.

Signature associée : **Toute votre ferme. Un seul flux.**

La signature peut apparaître sur les écrans marketing, landing page, README, onboarding et supports de présentation, mais ne doit pas surcharger l'interface applicative.

## Typographies

Typographie de marque :

- Manrope ExtraBold pour le logo ;
- Manrope Bold / SemiBold pour les titres marketing ;
- Manrope Medium pour les sous-titres.

Typographie d'interface :

- Inter ou Manrope ;
- priorité à la lisibilité ;
- chiffres bien lisibles dans les tableaux, KPI et dashboards.

Hiérarchie recommandée :

- H1 : Manrope Bold / ExtraBold ;
- H2 : Manrope Bold ;
- H3 : Manrope SemiBold ;
- texte courant : Inter Regular ou Manrope Regular ;
- labels UI : Inter Medium ou Manrope Medium ;
- KPI : Manrope Bold.

## Palette de couleurs

Palette principale :

- Bleu marine profond : `#0A132B`
- Bleu électrique : `#2563EF`
- Cobalt / indigo : `#4F46E5`
- Gris clair : `#F2F4F7`
- Blanc : `#FFFFFF`
- Texte principal : `#0F172A`
- Texte secondaire : `#64748B`
- Bordures : `#E2E8F0`

Couleurs fonctionnelles :

- Succès : `#10B981`
- Alerte : `#F59E0B`
- Danger : `#EF4444`
- Information : `#2563EF`

Le vert ne doit pas être la couleur principale de FarmFlow. Il peut être utilisé uniquement comme couleur fonctionnelle de succès ou indicateur positif.

## Style UI

L'interface doit être claire, fluide, aérée, professionnelle, moderne, lisible rapidement et orientée cockpit / pilotage.

Les écrans privilégient :

- cartes blanches ;
- fond gris très clair ;
- sidebar bleu marine ;
- accents bleu électrique ;
- KPI bien visibles ;
- graphiques sobres ;
- tableaux propres ;
- pictogrammes linéaires ;
- beaucoup d'espace blanc ;
- coins légèrement arrondis ;
- ombres très discrètes.

## Sidebar

La sidebar principale doit être en bleu marine profond. Elle contient le logo FarmFlow, la navigation principale, des icônes linéaires, l'état actif en bleu électrique, du texte blanc ou gris clair et un bas de sidebar avec exploitation / utilisateur.

Ne pas utiliser de sidebar verte.

## Composants

Bouton principal : fond bleu électrique, texte blanc, coins arrondis et hover légèrement plus sombre.

Bouton secondaire : fond blanc, bordure gris clair, texte bleu marine.

Cartes KPI : fond blanc, bordure gris clair, titre discret, valeur forte, variation colorée et icône légère.

Alertes : danger en rouge, avertissement en orange, succès en vert, information en bleu.

Graphiques : lignes bleues/cobalt, remplissages très légers, pas de couleurs criardes et pas de palettes vertes agricoles dominantes.

## Iconographie

Style : icônes linéaires, trait simple, coins arrondis, cohérence visuelle, pas d'illustrations complexes.

Icônes possibles : tableau de bord, parcelles, animaux, stocks, ventes, finances, météo, documents, paramètres, analyses.

## Ton marketing

FarmFlow parle comme un outil professionnel de pilotage agricole : clair, direct, sérieux, moderne, proche du terrain, sans jargon excessif, sans discours “bio” ou militant, orienté maîtrise, décision et performance.

Messages clés :

- Toute votre ferme. Un seul flux.
- Le cockpit numérique des fermes modernes.
- Centralisez vos données.
- Pilotez vos décisions.
- Anticipez vos besoins.
- Développez votre exploitation.

## Interdits visuels

Ne pas utiliser de vert dominant, feuilles partout, pictogrammes agricoles génériques, mascotte, logo complexe, arrière-plans ruraux clichés, typographies rustiques, textures bois/paille, style “ferme bio artisanale”, dégradés trop flashy, interfaces sombres partout ou dashboards surchargés.

## Règle obligatoire pour toute génération frontend

Avant de créer ou modifier une page frontend, vérifier que :

- la palette respecte cette identité ;
- la sidebar est cohérente ;
- les boutons utilisent les bons styles ;
- la typographie est homogène ;
- l'interface ne bascule pas vers une esthétique verte/bio ;
- les composants restent sobres, lisibles et professionnels.

Toute nouvelle page doit ressembler à une partie du même produit FarmFlow.
